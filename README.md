# Swift-Projects
A collection of swift apps made for learning ios development, projects explore the ios sdk and include the following:

* Animations
* Cat Year Calculator
* Controlling Keyboard
* Core Data
* Downloading Web Content
* JSON 
* Mapkit
* Permanent Storage
* Playgrounds
* Segueways
* Shake and Swipes
* Storing Images
* Parse
* Weather Checker
* Location Checker
* Audio
* Blog Reader
* Memorable Places
* Table Views
* Sound Shaker
* Tic Tac Toe
* Instagram Clone
* Snapchat Clone
